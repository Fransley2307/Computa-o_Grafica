# 🎨 Computação Gráfica

Este repositório contém um projeto desenvolvido para a disciplina de **Computação Gráfica**, utilizando a biblioteca **Pygame** em Python.  
O projeto demonstra conceitos básicos de animação, movimentação, colisão e renderização de textos em uma janela gráfica.
